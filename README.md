# BMI Calculator using Python

This is a command-line BMI Calculator developed using Python
as part of the OASIS INFOBYTE Internship Program.

## Project Description
The application calculates the Body Mass Index (BMI)
based on user-provided weight and height and classifies
the result into standard health categories.

## Technologies Used
- Python

## Features
- Accurate BMI calculation
- BMI classification (Underweight, Normal, Overweight, Obese)
- User-friendly command-line interface

## How to Run
1. Open terminal or command prompt
2. Run the program:
   python bmi_calculator.py
